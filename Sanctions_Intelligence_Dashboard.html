Operational recommendations

  1. Multi-source by default. No single list covers >81% of the universe; build the screening set from all four regimes.
  2. Tier by marginal contribution. OFAC SDN as spine, EU+UK for Russia/Ukraine, UN for legal baseline — not equal weight.
  3. Transliteration-aware matching. Apply Cyrillic/Arabic/Farsi/Han romanisation rules where false negatives are manufactured.
  4. Cross-source conflict rules. Define attribute precedence so the same subject is adjudicated identically regardless of source.
  5. Pin the version string. Store the OpenSanctions version (e.g. 20260517074701-kjc) with every alert disposition for replayable audit.
  6. Cadence to velocity. Tighten refresh on cyber and trafficking programmes; watch the UK feed's longer processing lag.
